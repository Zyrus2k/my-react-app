# Personal Profile Web Application

This is a personal profile web application built using React.js. It showcases various components including Home, About, Skills, and Contact, providing a platform to present personal information, skills, and a way for visitors to get in touch.

## Features

- **Home**: A welcoming introduction with career goals.
- **About**: Personal background, education, and work experience.
- **Skills**: A list of professional skills.
- **Contact**: A form for visitors to send messages.

## Technologies Used

- React.js
- React Router
- CSS for styling

## Getting Started

To get a local copy up and running, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/personal-profile-app.git
   ```

2. Navigate to the project directory:
   ```
   cd personal-profile-app
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```

5. Open your browser and go to `http://localhost:3000` to view the application.
